Épico 1 — Gestão de Leads 
História de Usuário 1 
Como operador, 
quero cadastrar leads automaticamente a partir das redes sociais, 
para evitar perda de oportunidades de atendimento. 
Definition of Done (DoD): 
● Lead criado automaticamente via integração. 
● Campos obrigatórios preenchidos (nome, telefone e canal). 
● Sistema impede cadastros duplicados. 
● Lead visível apenas para operadores autorizados. 
● Registro salvo no banco de dados. 
História de Usuário 2 
Como operador, 
quero visualizar e priorizar leads, 
para melhorar o tempo de atendimento. 
DoD: 
● Lista de leads exibida corretamente. 
● Possibilidade de filtro e ordenação. 
● Tempo até primeiro atendimento registrado. 
● Interface responsiva. 
● Dados carregados em até 3 segundos.

História de Usuário

● IMC calculado automaticamente. 
● Sistema bloqueia duplicidade. 
● Dados armazenados conforme LGPD. 
● Cadastro disponível para consulta imediata.
